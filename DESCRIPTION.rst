Python library for converting (and filtering) spectral data in various formats.

Supported formats:

* ADAMS (read/write)
* ARFF row-wise (read/write)
* ASC (read/write)
* ASCII XY (read/write)
* CAL FOSS (read/write)
* CSV (read/write)
* DPT (read/write)
* MPS (read)
* NIR FOSS (read/write)
* Opus Bruker (read)
* Opus Ext Bruker (read)
* SPA Thermo Fisher (read)
