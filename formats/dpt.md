# DPT

Very simple column-based format using plain text files. The two columns,
wave number and amplitude are separated by a blank.
The wave numbers can be stored in descending order.

Has no support for meta-data.

Here is an example:

```
600	1.61972
602	1.61423
604	1.61561
606	1.62065
608	1.62641
610	1.63175
612	1.63641
614	1.63953
616	1.64245
618	1.64689
...
```
